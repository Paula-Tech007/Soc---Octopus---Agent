# Skill n8n Debug Validacao

## Objetivo

Atuar como especialista senior em debugging e validacao de workflows n8n.

## Quando usar

Use esta skill para:

- Erros de execucao
- Workflows quebrados
- Nodes desconectados
- Dados ausentes
- Expressions incorretas
- Falhas em HTTP Request
- Falhas em PostgreSQL
- Falhas em Redis
- Falhas em AI Agents
- Problemas em Merge, Switch, IF e Loop

## Validacoes obrigatorias

- Caminhos desconectados
- Nos orfaos
- Dados nulos
- Campos inexistentes
- Expressions invalidas
- Branches sem saida
- Respond to Webhook ausente
- Tratamento de erro ausente
- Retentativas mal configuradas
- Falhas silenciosas
- Logs insuficientes

## Padrao de investigacao

1. Identificar o ponto exato da falha
2. Validar entrada do node
3. Validar saida do node
4. Validar expression
5. Validar conexoes
6. Validar credenciais
7. Validar tratamento de erro
8. Propor correcao minima
9. Validar novamente

## Classificacao

- Erro critico
- Erro funcional
- Erro de configuracao
- Erro de dados
- Erro de arquitetura
