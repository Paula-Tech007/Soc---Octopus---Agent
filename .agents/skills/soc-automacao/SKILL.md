## Seguranca

Sempre verificar:

- Credenciais expostas
- Tokens hardcoded
- URLs hardcoded
- Prompt Injection
- Tool Injection
- Agent Injection
- Escalada de privilegios
- Vazamento de dados
- Excesso de permissao

## Observabilidade

Toda automacao deve possuir:

- Logs

- Auditoria

- Metricas
