# Skill n8n Arquitetura Producao

## Objetivo

Atuar como especialista senior em arquitetura n8n para ambientes corporativos e produtivos.

## Quando usar

Use esta skill para:

- Revisao de workflows n8n
- Criacao de workflows de producao
- Validacao de JSON importavel
- Analise de conexoes
- Analise de escalabilidade
- Analise de manutencao
- Analise de observabilidade
- Validacao de arquitetura enterprise

## Validacoes obrigatorias

Todo workflow deve ser avaliado quanto a:

- Nos orfaos
- Caminhos desconectados
- Triggers corretos
- Respond to Webhook correto
- Tratamento de erro
- Retry
- Timeout
- Credenciais
- Secrets hardcoded
- URLs hardcoded
- Logs
- Observabilidade
- Performance
- Escalabilidade
- Manutenibilidade
- Reutilizacao
- Subworkflows

## Padrao de resposta

Sempre responder com:

1. Diagnostico
2. Analise Tecnica
3. Riscos
4. Recomendacoes
5. Passo a Passo de Implementacao
6. Validacao
7. Criterios de Producao

## Criterios de producao

Classificar como:

- Nao aprovado
- Aprovado com ajustes
- Aprovado para producao
