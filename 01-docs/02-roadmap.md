# Roadmap

## Fase 1 - Fundacao Documental E Estrutural

Objetivo: criar a estrutura base do projeto, documentar arquitetura inicial, riscos, roadmap e checklist.

Entregaveis:

- Pastas base do projeto.
- Visao geral.
- Arquitetura conceitual.
- Roadmap.
- Checklist da Fase 1.

Criterio de saida:

- Estrutura criada e validada.
- Documentos iniciais revisados.
- Aprovacao humana para iniciar a Fase 2.

## Fase 2 - Contratos E Payloads Mockados

Objetivo: definir contratos de entrada e saida usando apenas dados ficticios.

Entregaveis:

- Modelo conceitual de alerta.
- Modelo conceitual de ticket.
- Modelo conceitual de evento.
- Modelo conceitual de pergunta.
- Exemplos mockados.

## Fase 3 - Prompts Base Dos Agentes

Objetivo: criar prompts padronizados para o Cerebro Central e especialistas.

Entregaveis:

- Prompt do orquestrador.
- Prompts dos especialistas.
- Padrao obrigatorio de resposta.
- Regras de seguranca e confianca.

## Fase 4 - Fluxo Logico Multiagente

Objetivo: documentar o ciclo completo de analise, desde entrada ate saida consolidada.

Entregaveis:

- Fluxo de decisao.
- Matriz de roteamento por causa raiz.
- Criterios de escalonamento.
- Pontos de aprovacao humana.

## Fase 5 - Prototipo n8n Com Mocks

Objetivo: criar um primeiro workflow controlado no n8n usando dados ficticios.

Entregaveis:

- Workflow mockado.
- Nos de roteamento.
- Saida consolidada.
- Registro de execucao sem dados reais.

## Fase 6 - Persistencia Planejada

Objetivo: desenhar estrutura de persistencia futura.

Entregaveis:

- Modelo inicial MySQL/MariaDB.
- Uso previsto de Redis.
- Estrategia de auditoria.
- Campos para memoria corporativa futura.

## Fase 7 - Testes E Validacao

Objetivo: criar cenarios de teste controlados.

Entregaveis:

- Testes de alertas SOC.
- Testes de identidade.
- Testes de endpoint.
- Testes de rede.
- Testes de cloud.
- Testes de falso positivo.

## Fase 8 - Governanca E Memoria Corporativa

Objetivo: definir criterios para aprendizado, retencao e revisao humana.

Entregaveis:

- Politica de alimentacao da memoria.
- Criterios de confianca.
- Processo de revisao humana.
- Retencao e auditoria.
