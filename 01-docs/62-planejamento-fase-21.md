# Fase 21 - Registro De Decisao Para Versao Alvo Do n8n

## Objetivo

Criar um registro de decisao para a futura escolha da versao alvo do n8n, sem selecionar versao nesta fase e sem executar qualquer validacao em instancia real.

## Escopo

Artefatos cobertos nesta fase:

- `01-docs/64-registro-decisao-versao-n8n.md`
- `01-docs/14-guia-prototipo-n8n-mocks.md`
- `01-docs/28-indice-mestre.md`
- `01-docs/30-backlog-gates-decisao.md`
- `01-docs/61-checklist-operacional-importacao-n8n.md`
- Checklist da Fase 21.
- Relatorio e resumo do validador local.

## Fora De Escopo

Nesta fase nao serao alterados:

- Workflow n8n mockado.
- Payloads mockados.
- SQL planejado.
- Prompts.
- Diagramas.
- Validador estatico.

Nesta fase nao serao executados:

- Pesquisa externa de versao.
- Instalacao do n8n.
- Importacao no n8n.
- Workflow n8n.
- SQL.
- Banco.
- Redis.
- APIs.
- IA.
- Integracoes externas.
- Criacao de credenciais.

## Decisao Tecnica

A versao alvo do n8n permanece nao decidida.

O projeto passa a ter um registro formal para documentar, em fase futura:

- Versao candidata.
- Fonte da informacao usada para escolher a versao.
- Responsavel pela decisao.
- Ambiente de laboratorio.
- Criterios de compatibilidade.
- Riscos conhecidos.
- Resultado da aprovacao humana.

- Planejamento documentado.

- Guia n8n atualizado com referencia ao registro.

- Indice mestre atualizado.
- Relatorio JSON regerado.

- Checklist da fase atualizado.
- Projeto pausado aguardando aprovacao humana para nova fase.
