## Decisao Tecnica

O projeto passa a exigir um plano de evidencias para qualquer importacao futura.

Esse plano deve orientar:

- Evidencias antes da importacao.
- Evidencias durante a importacao.
- Evidencias apos a importacao.
- Comparacao entre o JSON aprovado e uma exportacao futura.
- Registro de divergencias.
- Criterios de aceitacao, bloqueio e rollback.
